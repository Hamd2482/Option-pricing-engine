# Option Pricing Engine (Monte Carlo + Black-Scholes)

## Overview

This project implements an option pricing engine using both analytical and simulation-based approaches.

## Features

* Black-Scholes analytical pricing
* Monte Carlo simulation for option pricing
* Greeks calculation (Delta, Gamma)
* Convergence analysis (Monte Carlo vs Black-Scholes)
* Error tracking and visualization
* CSV export for results

## Tech Stack

* Python
* NumPy, SciPy
* Matplotlib
* Pandas

## How to Run

```bash
pip install -r requirements.txt
python option_pricing_engine.py
```

## Output

* Option price (Black-Scholes vs Monte Carlo)
* Greeks
* Convergence plots
* results.csv file

## Learnings

* Stochastic modeling and simulation
* Numerical methods for pricing
* Importance of convergence in Monte Carlo methods
